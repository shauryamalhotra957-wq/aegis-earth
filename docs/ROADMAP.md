# Roadmap

## Production Milestone 1

- Add authenticated agency workspaces.
- Add incident audit logs.
- Add real geospatial layers through MapLibre or deck.gl.
- Add adapter interfaces for FEMA, NOAA, NASA FIRMS, local shelters, hospitals, and road closures.

## Production Milestone 2

- Human approval workflow for recommended interventions.
- Model cards for every scoring function.
- Bias and fairness reports by district.
- Offline mobile responder mode.

## Production Milestone 3

- Multi-agency simulation exercises.
- Live logistics inventory.
- Post-incident learning loop.
- Formal safety case and emergency-management certification pathway.
