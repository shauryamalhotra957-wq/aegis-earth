# Testing

The test suite focuses on the deterministic engine because that is where a polished demo can quietly become unsafe.

Run:

```bash
npm test
npm run build
```

Covered behavior:

- Risk increases when hazard, exposure, and signal pressure increase.
- Higher vulnerability regions receive fairness-aware priority boosts.
- The optimizer respects budget and team constraints.
- The scenario simulator improves outcomes when meaningful interventions are selected.
- Sanitization escapes risky text before export.
