# Demo Script

## 0:00 - Open

"This is Aegis Earth, a disaster resilience command center. The goal is not to visualize crisis data. The goal is to turn crisis data into an explainable plan."

## 0:30 - Risk Map

"Each region is scored from hazard, exposure, infrastructure fragility, live signal pressure, and equity need. The map is animated because the model is continuously recomputing state, not presenting a static report."

## 1:15 - Triage

"Ganges Delta ranks first in this scenario because flood, storm, heat, population exposure, and vulnerability stack together. Notice the confidence score: the system tells us how much trust to place in the decision."

## 2:00 - Intel Cell

"The right panel explains the decision. It does not just say 'high risk'; it shows dominant drivers and mission picks with impact, ETA, and rationale."

## 2:45 - Scenario Lab

"If I switch from Baseline to Shock, the simulator changes expected lives protected, sheltered population, unmet need, and risk reduction. This makes the system useful for planning before conditions deteriorate."

## 3:30 - Equity Guardrail

"The optimizer does not only chase the easiest wins. It checks coverage for vulnerable regions, so the plan can be defended ethically, not only mathematically."

## 4:15 - Export

"The command center exports an executive brief that can go directly into a situation room. The export path sanitizes text and the project includes a documented threat model."

## 5:00 - Proof

"This is a working React and TypeScript app. It has deterministic engine tests, UI tests, production build verification, and a clean dependency audit."
